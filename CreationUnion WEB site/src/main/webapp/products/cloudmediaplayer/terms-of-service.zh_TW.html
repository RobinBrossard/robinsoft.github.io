1. 服務說明

本應用程式是一款多媒體播放與管理工具，允許使用者連結自己擁有的第三方雲端儲存帳號，例如 Google Drive、OneDrive、Dropbox 與百度網盤。

本應用程式使用官方 API 掃描使用者授權的目錄，並將影片、圖片與音訊等媒體資源同步到本機快取以供播放。

2. 帳號與授權

第三方帳號

您理解並同意，本應用程式僅作為存取第三方雲端帳號的技術工具。您必須確保自己具有合法使用這些帳號的權利。

權限範圍

應用程式請求的權限（例如 .../auth/drive）僅用於掃描媒體清單、讀取中繼資料，以及將檔案下載到本機私有快取目錄。

使用者責任：您需對透過本應用程式存取或播放的所有內容承擔全部法律責任。

3. 資料使用與隱私

  • 本機處理：本應用程式下載的所有媒體資料僅儲存在您裝置的本機快取中，以提升播放效能並支援離線存取。
  • 不上傳原則：本應用程式絕不會將您的雲端檔案、中繼資料或個人登入憑證上傳至任何第三方伺服器。
  • 快取管理：本應用程式提供自動清理機制，使用者也可以隨時手動清除本機快取。

服務可用性

內容風險
