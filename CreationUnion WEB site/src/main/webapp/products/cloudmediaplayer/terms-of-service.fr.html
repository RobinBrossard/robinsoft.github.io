1. Description du service

L'application est un outil de lecture et de gestion multimédia qui permet aux utilisateurs de connecter leurs propres comptes de stockage cloud tiers, tels que Google Drive, OneDrive, Dropbox et Baidu Netdisk.

L'application utilise les API officielles pour analyser les répertoires autorisés par l'utilisateur et synchroniser les ressources multimédias, notamment les vidéos, images et fichiers audio, vers un cache local pour la lecture.

2. Comptes et autorisation

Comptes tiers

Vous comprenez et acceptez que l'application agit uniquement comme un outil technique pour accéder à vos comptes cloud tiers. Vous devez vous assurer que vous avez le droit légal d'utiliser ces comptes.

Portée des autorisations

Les autorisations demandées par l'application, telles que .../auth/drive, sont utilisées uniquement pour analyser les listes multimédias, lire les métadonnées et télécharger les fichiers dans un répertoire de cache privé local.

Responsabilité de l'utilisateur : vous assumez l'entière responsabilité juridique de tout contenu consulté ou lu via l'application.

3. Utilisation des données et confidentialité

  • Traitement local : toutes les données multimédias téléchargées par l'application sont stockées uniquement dans le cache local de votre appareil afin d'améliorer les performances de lecture et de permettre l'accès hors ligne.
  • Principe de non-téléversement : l'application ne téléversera jamais vos fichiers cloud, métadonnées ou identifiants de connexion personnels vers des serveurs tiers.
  • Gestion du cache : l'application fournit un mécanisme de nettoyage automatique et les utilisateurs peuvent également vider manuellement le cache local à tout moment.
